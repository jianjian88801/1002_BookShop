# 基于javaweb叮当书城

#### 介绍
基于javaweb叮当书城，前端采用jsp+layui，主要分两端Web端与管理后端，代码规范整洁，技术选型也非常经典，合适初学java编程者学习以及电商系统的入门。


#### 软件架构
前端：jsp | jquery | layui  
后端：spring | springmvc | mybatis  
环境：jdk1.7 | mysql | maven
